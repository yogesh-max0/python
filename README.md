# 🐍 Python Basics for Beginners

Welcome to the **Python Basics for Beginners** repository!

This repository contains beginner-friendly Python programs covering the fundamental concepts of Python programming. Each file focuses on a specific topic with simple examples to help beginners understand Python from scratch.

## ▶️ How to Run

Clone the repository:

```bash
git clone https://github.com/your-username/python-basics.git
```

Go to the project directory:

```bash
cd python-basics
```

Run any Python file:

```bash
python filename.py
```

Example:

```bash
python functions.py
```

## 📖 Learning Path

It is recommended to learn the topics in the following order:

1. Variables & Keywords
2. Data Types
3. User Input
4. Operators
5. Strings & Indexing
6. Conditional Statements
7. Loops
8. Functions
9. Collections
10. File Handling
11. Error Handling
12. Object-Oriented Programming (OOP)
